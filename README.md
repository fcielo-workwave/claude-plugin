# fra

Personal Claude Code plugin with skills for brainstorming and planning.

## Skills

- `/fra:brainstorm` — Explores your idea through collaborative dialogue, produces a design spec
- `/fra:write-plan` — Turns a spec into a detailed, step-by-step implementation plan

## Install

```shell
/plugin marketplace add fcielo-workwave/claude-plugin
/plugin install fra@fra-marketplace
```
